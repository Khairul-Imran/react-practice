Key Concepts Demonstrated with Todo List:

1. DOM Manipulation

- Modifying elements (classList, textContent)

2. Event Handling
- Form submission
- Click events
- Change events


3. State Management
- Keeping track of todos array
- Updating UI based on state changes


4. Local Storage
- Saving data (setItem)
- Loading data (getItem)
- JSON parsing/stringifying


5. Array Methods
- push
- find
- filter


6. Data Attributes
- Using dataset to store IDs
- Selecting elements by data attributes
